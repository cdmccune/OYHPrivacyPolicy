Collecting Information

This Application will collect the following information from the user: first/last name, email, password, names of scripture lists created, and scriptures added to each list.

This information will be collected only by the what the user inputs into the app by interaction with it. The personal information through creating an account and the scripture/list information through users adding them.

Use of Collected Information

There are several ways that the app uses this information.

1) Use of personal information to log the user in and authenticate their account.
2) Storage of scripture related information, stored on the cloud to allow for access from different devices.
3) Anonymous recording of saved scriptures in order to show the most popular books of the bible that have stored scriptures among the users. When a user saves or removes a scripture, it simply increments the count and doesn't store the users information.

Third Party Data Sharing

The Firebase Authentication (for user login) and FireStore (for cloud data storage) are used to aid in the user experience. A user's password is encrpyted as laid out in their privacy policy
